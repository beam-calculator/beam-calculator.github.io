Questions frequentes
Why is portee-focused bois sizing different from acier sizing?

Because bois users often start avec the practical question of how far a poutre can portee before fleche becomes uncomfortable or unacceptable, while acier users more often begin from a chosen section.

Does the bois portee page account pour long-term bois effects automatically?

It gives the elastic response engine pour preliminaire checks. Final bois decisions still need the appropriate long-term et project-specific adjustments.

Which page should I use if I already know the exact bois size?

Open the custom bois poutre size page if the section is the main starting point, or the Eurocode 5 bois page if you want a more standard-based framing.
