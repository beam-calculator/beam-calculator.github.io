Questions frequentes
Why is the formulas page not the top resultat here?

Because the intent is already more complex than one closed-form textbook setup. The dedicated three-point-charge solveur is built pour that actual charge pattern.

Can I approximate three point charges avec one equivalent charge?

Only as a rough shortcut. If the search already specifies three point charges, the distribution et locations matter enough that the dedicated page is the better answer.

What if one of the three charges is movable?

Use the dedicated page pour fixed locations first. If the design question becomes influence-based or moving-charge driven, that is a different analysis problem than the actuel poutre pages target.
