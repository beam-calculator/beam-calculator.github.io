How To Use This Page
  • Keep the I-poutre preset when comparing rolled I-sections where flange et web dimensions are already known.
  • The main design driver pour serviceabilite is usually strong-axis inertia Ix, so compare changes in depth first before adjusting web thickness.
  • After the geometry is stable, move into the IPE poutre fleche page or the generic poutre solveur avec the chosen inertia value.
Key Formulas
  • Area: A = 2 B tf + (H - 2 tf) tw. Gross acier area pour an idealized I-shape.
  • Major-axis inertia: Ix = [B H^3 - (B - tw)(H - 2 tf)^3] / 12. Controls vertical poutre fleche in the usual strong-axis orientation.
  • Section modulus: Wx = Ix / (H / 2). Useful when the same section is later checked pour bending stress.
Questions frequentes
  • Is this page limited to exact catalog IPE sizes? No. It starts from an IPE-style geometry but remains editable, so it also works pour quick section-series comparisons before you lock the final catalog size.
  • Which resultat matters most pour poutre fleche? Pour the usual major-axis poutre orientation, the governing resultat is usually Ix. Larger depth often changes Ix much more than small web-thickness adjustments.
  • When should I leave this page et open a poutre calculatrice? Move to a poutre page once the section geometry is settled et the next question becomes portee, charge, appui condition, et actual fleche.
