Assumptions & Limits

  • The model represents 2D bending response only et does not include torsion.
  • Appuis are idealized as analytical boundary conditions.
  • Construction stages, nonlinearity, et settlement effects need a more detailed model.

Hypotheses du modele
  • The poutre has two interior simple appuis et gratuit overhang segments.
  • The actuel setup combines one point charge avec an optional full-portee uniform charge.
  • Elastic Euler-Bernoulli poutre behavior is assumed.
Outils associes
  • Use this page when standard one-portee calculators no longer fit because the poutre extends beyond one or both appui lines.
  • Appui positions are editable, so users can test different overhang lengths without leaving the page.
  • If the real structure includes partial restraint, continuous framing, or nonlinear appui behavior, this remains a preliminaire poutre model.
