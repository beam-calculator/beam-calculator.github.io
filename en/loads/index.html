Module Scope
  • Use this section when the main question is how the load is formed, distributed, or combined before member sizing starts.
  • These tools are useful early on when slab loads must be converted into line loads or when concentrated loading patterns govern the response.
  • For final beam deflection after the load model is known, continue into the generic beam solver or one of the advanced beam pages.
