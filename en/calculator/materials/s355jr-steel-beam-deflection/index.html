Engineering Notes
  • For serviceability checks, S355JR uses essentially the same elastic modulus as standard S235 structural steel, so the beam stiffness still depends mainly on section geometry.
  • Use this page when the steel grade is fixed before the exact profile has been selected.
  • If the project is strength-driven, the higher yield level matters; if it is deflection-driven, changing the section geometry usually has more effect than changing between S235 and S355JR.
